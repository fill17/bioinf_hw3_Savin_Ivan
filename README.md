# bioinf_hw3_Savin_Ivan

Данная работа посвящена анализу RNA-seq данных мышиных эмбриональных фибробластов (MEFs) с целью выявления генов, изменяющих свою экспрессию в процессе перепрограммирования. В ходе выполнения задания были обработаны шесть образцов - три перепрограммированных и три контрольных, проведено выравнивание чтений на референсный геном mm10, оценка качества данных и подсчет количества чтений на гены. Полученные результаты позволяют идентифицировать ключевые гены, вовлеченные в процесс клеточного перепрограммирования.
