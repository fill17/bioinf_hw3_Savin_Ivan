# bioinf_hw3_Savin_Ivan

файл alignment_statistics (1) содержит
Таблицу со статистикой по каждому из 6-ти образцов:
ID образца
Тип образца (перепрограммированние или контроль)
Общее кол-во исходных чтений
Кол-во и процент чтений, которые были успешно откартированы на геном (уникально или нет)
Кол-во и процент уникально откартированных чтений
Общее кол-во чтений, которые попали на гены

файл ALL.counts  -- сводная таблица, где указано кол-во чтений уникально картированных на каждый ген в каждом образце

гугл колаб SAvin_Ivan_BIO_Infa.ipynb содержит весь код

гугл колаб статистика.ipynb содержит статистику fastq 

анализ RNA-seq данных мышиных эмбриональных фибробластов (MEFs) с целью выявления генов, изменяющих свою экспрессию в процессе перепрограммирования. В ходе выполнения задания были обработаны шесть образцов - три перепрограммированных и три контрольных, проведено выравнивание чтений на референсный геном mm10, оценка качества данных и подсчет количества чтений на гены. Полученные результаты позволяют идентифицировать ключевые гены, вовлеченные в процесс клеточного перепрограммирования.
